

# r2Vault — Landing Page

A sleek, modern marketing page to showcase your macOS R2 storage client.

## Pages & Sections

### Single-page landing (`/`)

1. **Hero Section** — Bold headline ("Your R2 Bucket, Beautifully Managed"), subtitle, animated gradient background, two CTAs: "Download for macOS" (links to latest GitHub release) and "View on GitHub". A hero mockup/screenshot of the app.

2. **Features Grid** — 6 feature cards with icons highlighting:
   - Drag & Drop Uploads with progress tracking
   - File Browser with grid/list views & thumbnails
   - Presigned URL sharing
   - Upload History & Queue management
   - Multiple Bucket support
   - Menu Bar quick access

3. **Screenshot Showcase** — Large app screenshot with a macOS-style window frame, subtle shadow and glow effects.

4. **Tech Stack** — Minimal section showing SwiftUI, MVVM, async/await, CryptoKit badges.

5. **Getting Started** — 3 simple steps: Download DMG → Configure credentials → Start managing files.

6. **Footer** — GitHub link, MIT license badge, version (v1.2.1).

## Design Direction
- Dark theme by default (fits developer tooling aesthetic)
- Accent color: Cloudflare orange (#F6821F) paired with deep navy/slate backgrounds
- Smooth scroll animations on section entry
- Clean typography, generous whitespace
- Responsive for all screen sizes
